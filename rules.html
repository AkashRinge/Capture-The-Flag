General Rules and Hints


  • Guess what? There are no rules to how this game has to be played


  • The only objective is to reach the next level


  • Try finding hints all throughout the webpage, try manipulating the webpage link and all sorts of experiments possible with the webpage to obtain hints.


  • When you manipulate links, use ONLY AND ONLY small case alphabets. You will know for certain when you crack a solution.


  • You might need to use the internet for some levels. Mostly a google search bar will be provided for such levels. For the rest others, it is possible to crack the solution without the help of the internet too


  • Keep saving the web links (to a word file or something) to each level. They are necessary not only for the coming levels, but also for us to declare the winner.


  • Since the game requires manipulations, don't use a mobile device to play.


  • Please do not disclose links to others, it spoils the spirit of the game. If you really appreciate our efforts, help us by only helping yourself. Bragging about cracking levels does not make any sense


  • Don't illegaly jump levels or you'll be banned.


  • The final authority lies with the creators of this game.
